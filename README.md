# Ejercicio-Practico-API-Rest
Aplicación web diseñada para interactuar con la API de Studio Ghibli, demostrando habilidades en el manejo de peticiones asíncronas, procesamiento de datos en formato JSON y manipulación dinámica del DOM.

# Demo
https://abomdevp.github.io/Ejercicio-Practico-API-Rest/

Funcionalidades
- Consumo de servicios externos mediante Fetch API.
- Filtrado dinámico de información basado en la selección del usuario.
- Presentación de datos específicos: Títulos, directores, años de lanzamiento y descripciones.
- Visualización de resultados en tiempo real tanto en la interfaz de usuario como en la consola del desarrollador.
- Manejo de estados de carga y visualización de datos mediante listas dinámicas.

Uso
1. Clonar o descargar el proyecto
2. Abrir index.html en el navegador
3. Listo (no requiere instalar dependencias)
